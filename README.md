# GREEN-API Test Page

Простая страница для тестирования WhatsApp API через GREEN-API.

## Что умеет

- getSettings — получить настройки инстанса
- getStateInstance — проверить статус авторизации
- sendMessage — отправить текстовое сообщение
- sendFileByUrl — отправить файл по ссылке

## Как использовать

1. Зарегистрируйся на [green-api.com](https://green-api.com/) и создай инстанс
2. Привяжи WhatsApp через QR-код
3. Открой `index.html` в браузере
4. Вставь `idInstance` и `ApiTokenInstance`
5. Готово, можно тестировать методы

## Стек

HTML + CSS + vanilla JS (fetch API)
